npm(node package manager) - node.js의 패키지를 관리하는 도구
package.json 파일을 통해 프로젝트 정보와 패키지의 정보를 관리한다.
npm install에는 지역과 전역이 있는데, 별도의 언급이 없으면 지역으로 설정되며,
전역으로 다운롣드 시 npm install -g 를 하면 된다.

npx 지역 또는 전역에 패키지를 설치하지 않고 최신 버전의 패키지를 가져와 실행할 수 있다.
package-lock.json은 설치 시점 당시의 읜존성 트리 정보를 저장하고 있으며,
이 파일이 있다면 npm은 이 파일의 의존성 트릴르 기준으로 패키지를 설치한다.
